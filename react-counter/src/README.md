# 🚀 React Counter App

A simple and responsive Counter Application built with **React** and **Vite**. This project demonstrates the fundamentals of React, including components, state management using the `useState` hook, event handling, and component-based styling.

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd react-counter
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
